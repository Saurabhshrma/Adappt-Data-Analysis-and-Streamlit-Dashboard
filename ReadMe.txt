TO RUN THE STREAMLIT DASHBOARD :

1.Make sure all the contents are present in this folder- 
  - styles.css
  - Adappt.png
  - filtered_df.xlsx

2. Make sure you have already installed streamlit on you     device.

  - use this command to install 'pip install streamlit'

3. After the basic checks open your cmd

4. Change directories to the place where this folder is present.

5. Make sure you are inside Adappt Folder where Dashboard.py is present. It will run the streamlit webapp.

6. Run command :
   streamlit run Dashboard.py

Dashboard will be visible
